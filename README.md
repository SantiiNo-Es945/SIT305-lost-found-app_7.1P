# SIT305 Task 7.1P – Lost and Found App

## Overview
This application allows users to post lost and found items. 
Each advert includes a description, image, category, location, and an automatically generated date and time.

## Features
- Create new lost or found adverts
- Upload an image for each item
- View all adverts in a list
- Search/filter adverts by category
- View detailed information for each advert
- Remove adverts when items are found

## Technologies Used
- Java
- Android Studio
- SQLite
- RecyclerView

## How it works
The app stores all adverts locally using SQLite. Users can create adverts, which are saved into the database.
The adverts are displayed using a RecyclerView, and users can filter them by category or view full details and remove them.

## How to run the app
Open the project in Android Studio and run it on an emulator or phone.
